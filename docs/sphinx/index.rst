
HAL's documentation
===================

This is the documentation for the SDSS Python product HAL. The current version is |hal_version|. You can install the package by doing

.. code-block:: console

  $ pip install sdss-hal

.. warning ::
  HAL is the codename of the software product. When connected to Tron, the actor commands are associated with the ``sop`` actor (e.g., ``sop gotoStow``).
